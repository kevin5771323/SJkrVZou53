# 酒店后台管理系统 java985

## 项目概述

酒店后台管理系统是一款基于Vue3和Nodejs的前后端分离的系统。它通过高效的后端服务和友好的前端界面，提供了一套完整的酒店管理解决方案。

## 技术栈

### 前端

- Vue3
- Element-Plus
- Axios
- ECharts
- Pinia
- TypeScript

### 后端

- Nodejs
- Express

### 数据库

- MySQL 5.7/8.0

### Nodejs版本

- v18.20.4

## 功能模块

- 登录注册
  - 用户登录
  - 用户注册
- 角色管理
  - 角色权限分配
- 用户管理
  - 用户信息管理
- 房型管理
  - 房型信息维护
- 房间管理
  - 房间状态管理
- 入住用户管理
  - 入住信息记录
- 个人信息管理
  - 修改个人信息
  - 修改密码

## 系统角色

- 系统管理员
- 酒店经理
- 前台接待

## 运行环境

- 前端：现代浏览器
- 后端：支持Nodejs的环境
- 数据库：MySQL 5.7/8.0

## 核心亮点

- 基于Vue3，使用TypeScript开发，确保代码的可维护性和健壮性。
- 使用Element-Plus UI库，界面友好，易于操作。
- 前后端分离的架构设计，便于系统的维护和扩展。
- 支持多角色登录，满足不同职责的管理需求。
